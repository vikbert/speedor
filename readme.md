## Starting started
```bash
$ npm install --global speedor
$ speedor
```


## Usage

```
$ speedor --help

  Usage
    $ speedor

  Options
    --json -j     Output the result as JSON
    --bytes -b    Output the result in megabytes per second (MBps)
    --verbose -v  Output more detailed information
```
